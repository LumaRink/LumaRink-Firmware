# LumaRink Firmware

This repository contains the pre-installed firmware for LumaRink 5x5 LED matrix boards.

## Features
- 5x5 RGB LED matrix
- WiFi-enabled (optional NHL API for hockey updates)
- Pre-flashed on barebones and assembled boards
- Open-source firmware for customization or recovery

## Getting Started
No programming required for standard use.  
See /docs/flashing.md for instructions if you need to re-flash the firmware.
